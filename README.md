# VindexValor

Open-source personal finance and budget management web application.

## Description

VindexValor is a web application focused on personal finance management, budgeting, and transaction tracking.

The project is currently in active development and is being developed as a final course project (TCC) focused on building a long-term open-source platform.

At the moment, the application is still experimental and may contain bugs, incomplete features, and breaking changes.

Current features include account management, transaction tracking, dashboards, authentication, and cloud synchronization through Supabase.

Future plans include desktop and mobile applications, offline support, advanced analytics, investiments syncing, and multi-user budgeting.
